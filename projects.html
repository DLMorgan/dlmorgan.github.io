Wi-Drive

A Bluetooth controlled Robo-Car with Wi-Fi streaming capabilities. This purpose of this project is to use the speed advantage gained by connecting two phones via Wi-Fi Direct to stream one phones video camera to another phone in the nearby area. In addition, the streaming phone is used to control another device via bluetooth that allows a remote control car to drive around the room.

Chilipepper Labs

The Chilipepper Labs are a set of tutorials meant to show the workflow used to create a design on the Xilinx Zed board FPGA. The primary goal of these laboratory exercises is to demonstrate a flow that uses Mathworks HDL Coder with MATLAB to HDL code generation, Xilinx tools, and the Toyon Chilipepper board. The actual design created within the tutorials is a simple qpsk transmitter/reciever with a simple software defined data link layer

Indoor Localization

The Indoor Localization project is concerned with providing GPS-like localization services to users within an environment where GPS is unavailable. It primarily uses Wi-Fi and the onBoard phone sensors to accomplish this, and attempts to fuse the two data sources in a way that is constructive, and thus improves the idividual accuracies of both sources.

Kalman Based Heading estimation



The heading estimation project is an expansion on the previously mentioned Indoor Localization. It's primary purpose is to improve the orientation estimation provided by android for the specific case in which there are magnetic distortions in the nearby area which cause the magenetometer readings to be inaccurate.
